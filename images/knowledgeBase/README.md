## Model Characterization
This knowledge base contains trained AI models for German Credit Risk Classification:

- **ANN Model (currentAiSolution.h5):** Deep neural network with 4 hidden layers (128→64→32→16 neurons) achieving 76.36% test accuracy
- **OLS Model (currentOlsSolution.pkl & .xml):** Ordinary Least Squares regression model for baseline comparison
- **Task:** Binary classification of credit risk (good/bad)
- **Features:** 9 numerical features derived from German credit data
